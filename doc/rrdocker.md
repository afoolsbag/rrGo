## [使用](https://docs.docker.com/get-started/)

*   展示 Docker 帮助
    *   `docker --help`
    *   `docker <COMMAND> --help`
*   测试 Docker 版本
    *   简略信息：`docker --version`
    *   详细信息：`docker version`
    *   系统信息：`docker info`
*   测试 Docker 安装
    *   测试运行：`docker run hello-world`
    *   列出镜像：`docker image ls`
    *   列出容器：`docker container ls -a`
